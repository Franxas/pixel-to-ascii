Vídeo -> ascii transcription

Quick exercise where I developed a web application that captures a video feed from the computer's camera and transcribes it to a string with an even screen distribuition based on the captured pixels.

-> capturing the camera's signal.  
-> downsampling the video.  
-> make a reading of the pixel's values.  
-> replace each pixel with a selection of characters (ascii / emojis) based on those values.  

Controls:  
-> run it with node  

-- browser window --  
-> choose a file from your computer and use that instead of webcam  
-> slider ajusts brightness  
-> snapshot buttobn will print a snapshot in your shell
